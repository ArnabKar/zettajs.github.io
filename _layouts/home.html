Run Everywhere

Build with APIs

Zetta provides powerful Siren hypermedia APIs that allow you to model device and application state as REST interfaces so that you can easily design an API-first architecture for your Internet of Things projects.

server
 .observe([sound, arm, huelight],

   var gocrazy = new GoCrazy(arm, huelight);
   sound.streams.level.on('data', function(msg) {

 });
