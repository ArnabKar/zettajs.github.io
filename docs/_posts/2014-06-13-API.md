####Peer Linking


`GET http://zetta-cloud.herokuapp.com/admin`

```
{
  "class": ["admin"],
  "actions": [
    {
       "name": "link",
       "method": "POST",
       "href": "http://zetta-cloud.herokuapp.com/admin/peers",
       "fields": [
         { "name": "url", "type": "url" }
       ]
    }
  ],
  "link": [
    {
      "rel": ["self"],
      "href": "http://zetta-cloud.herokuapp.com/admin"
    },
    {
      "rel": ["http://rels.zettajs.io/peers"],
      "href": "http://zetta-cloud.herokuapp.com/admin/peers"
    }
  ]
}
```
