Zetta is an API-First Development Platform for the Internet of Things.

ZettaÂ® is an API-first, open source development platform built on Node.js for easily building Internet of Things applications that run across geo-distributed and cloud computers. Zetta combines Siren hypermedia APIs and WebSockets with reactive programming, perfect for assembling many devices into big data-intensive, real-time applications.
